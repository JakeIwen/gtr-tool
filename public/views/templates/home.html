GTR Tools

GTR Tools is a is a feature-packed web application wih many useful tools for aspiring guitarists and singer-songwriters. Initially developed to help guitarists better learn the many fingerings of basic three-note chords (triads), it has been expanded on to include more complex chords and scales. This app aims to provide tools that other online music tools do not, and as such, some understanding of basic theory may be beneficial to the user.

CAGED+ Mapper

The five most common movable major chord shapes that can be played anywhere on the fretboard are the C, A, G, E and D shapes. The CAGED+ mapper was designed to allow one to view the location and notes of these five shapes for a musical key of their choosing

The tool has been expanded on to allow the viewing of full scales, including the seven musical modes as well as pentatonic scales.

Triad+ Tool

The Triad+ tool will display variations of numerous ways to play a particular chord. Use the filters to narrow down the number of matching variationsto help yourself discover new chord shapes. Key bindings are shown square brackets near their respective filters.


Filters

  • Inversion fiter: View more formations containing non-tonic notes as the lowest note (inversions).
  • Max Fret-Stretch: Set this to prevent the dispaly of chords having a span that exceeds your hand's ability to stretch
  • Open Strings: Enable this to alllow chords containing open strings even if it exceets the Max Fret-Stretch
  • Consecutive Strings: Disable to dispaly chrods that include one or more unused strings between the used strings
  • Octave duplicates: Disable this to show variations with only the exact number of notes in the chord
  • String Selectors: The clicking a blue string selector will prevent the dispaly of chords using that string.
  • Fretboard Section: The double slider bar selects the area of the fretboard in which chords will be displayed.

Text-Transposer

The Text Transposer is designed to identify chords in in text documents containing song lyrics and chords. Paste or type the text and chords ofr a song of your choice to use transpose buttons, changing the key of the song by one semi-tone at a time.
